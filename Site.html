Frameworkler

React JS

React kullanıcı arayüzü oluşturmaya yarayan açık kaynak kodlu bir JavaScript kütüphanesidir. Facebook tarafından geliştirilmiştir ve Model-View-Controller prensibine uygundur.

Angular JS

Angular, Google tarafından desteklenen, dinamik web uygulamaları oluşturmak için yazılmış bir JavaScript frameworküdür.

Vue JS

Vue.js kullanıcı arayüzleri ve tek sayfa uygulamalar için açık kaynaklı bir frameworkdür. Basit ve esnek bir yapı sunar.

Svelte

Svelte, hızlı ve verimli bir frameworkdür. Kullanıcıların ihtiyaçlarını optimize bir şekilde karşılar.

Polymer JS

Polymer, Google tarafından geliştirilmiş, web bileşenleri kullanarak uygulama geliştirmek için kullanılan bir JavaScript kütüphanesidir.

Ember JS

Ember.js, karmaşık web uygulamaları oluşturmak için geliştirilen bir frameworkdür. Otomatik güncelleme sağlar.
